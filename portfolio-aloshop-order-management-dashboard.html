Alodokter Shop Order Management Dashboard

Project Overview

  • CompanyAlodokter
  • RolePrincipal Architect Fullstack
  • Year2024

Project Description

Developed a web-based dashboard solution for Alodokter's pharmacy partners to efficiently manage and process orders from Alodokter app users.

This system serves as a critical operational tool connecting the healthcare platform with its pharmacy network.

Key Responsibilities

Product Development
  • Led the development of a comprehensive order management dashboard tailored for pharmacy partners.
  • Collaborated closely with the Product Team during discovery phases to enhance the Product Requirement Document (PRD) with technical and infrastructure insights.
  • Designed an intuitive interface allowing pharmacy partners to track, process, and fulfill medication orders.
Technical Architecture
  • Created detailed architectural blueprints for the entire system.
  • Designed integration patterns between various systems and services.
  • Implemented a scalable MongoDB database structure optimized for order management workflows.
  • Utilized Socket.io for real-time order notifications and updates.
  • Implemented Redis for caching and improving dashboard performance.
  • Set up Sidekiq for background job processing to handle order fulfillment tasks.
Infrastructure & Deployment
  • Established the complete service ecosystem from development to production.
  • Configured and deployed the application in Google Kubernetes Engine (GKE).
  • Implemented containerization using Docker for consistent deployment across environments.
  • Optimized cloud resources on Google Cloud Platform (GCP) for cost-effectiveness and performance.
